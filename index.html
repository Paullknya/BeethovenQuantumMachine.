🎵 Beethoven Quantum Machine

Эмулятор 8-кубитного квантового компьютера с консольным интерфейсом, вентилями X, H, Z, CNOT, поддержкой шумов и собственным установщиком для Windows.

⬇️ Скачать для Windows (v1.5)

(Ссылка появится после создания релиза GitHub)

📋 Быстрая справка


h    — вентиль Адамара: создаёт суперпозицию
x    — вентиль NOT: переворачивает кубит
z    — вентиль Паули Z: меняет фазу |1⟩ на -|1⟩
cnot — управляемое NOT (запутывание)
measure   — измерить все 8 кубитов
reset     — сбросить состояние
cls       — очистить экран консоли
exit      — выйти

🎛️ Запуск с шумом (имитация реального процессора)

BeethovenQuantumMachine.exe --noise 0.1
