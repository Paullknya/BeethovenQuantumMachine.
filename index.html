🎵 Beethoven Quantum Machine

Квантовый эмулятор 8 кубитов v1.5
⬇️ Скачать для Windows

🔹 Вы перейдёте на страницу релиза → в разделе Assets выберите файл
BeethovenQuantumMachineSetup.exe или ZIP-архив. Скачивание начнётся сразу.

💡 Не открывается скачанный файл?
Если Windows показывает предупреждение — нажмите «Подробнее» → «Выполнить в любом случае»
Или: правой кнопкой по файлу → «Свойства» → внизу «Разблокировать» → ОК → запустить снова.

📋 Быстрая справка


h        — вентиль Адамара: создаёт суперпозицию
x        — вентиль NOT: переворачивает кубит
z        — вентиль Паули Z: меняет фазу |1⟩ на -|1⟩
cnot     — управляемое NOT (запутывание кубитов)
measure  — измерить все 8 кубитов
reset    — сбросить состояние в |00000000>
cls      — очистить экран консоли
exit     — выйти из программы

🎛️ Запуск с шумом (реалистичный режим)

BeethovenQuantumMachine.exe --noise 0.1

— вероятность ошибки 10% (битовые флипы, шум чтения). Имитация реального квантового процессора.

📦 Альтернативная установка: если .exe не запускается, скачайте ZIP-архив из того же релиза,
распакуйте и запустите BeethovenQuantumMachineSetup.exe.
